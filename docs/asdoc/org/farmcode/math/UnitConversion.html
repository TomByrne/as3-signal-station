public static function breakdown(from:Number, fromType:Number, toTypes:Array):Array

Converts a number in to an array representing its different base values

Parameters
from:Number — The number to convert
 
fromType:Number — The base of the number from. Use the NUMBER constants to represent this
 
toTypes:Array — An array of bases to convert to

Returns
Array — An array of numbers. The index of each returned number will correspond to the indicies of the given toTypes array
